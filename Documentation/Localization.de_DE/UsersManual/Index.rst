Benutzerhandbuch
================

Im BE müssen Sie nur den Adress Datensatz (tt_address) auswählen, der innerhalb
des Kalendertermins angezeigt werden soll.

Die erforderlichen Änderungen für die Kalendertermin Vorlage sind im Abschnitt
:ref:`my-templating-label` beschrieben.

Wenn Sie einen Link zu einer Detailseite, zum Beispiel mittels wt_directory, hinzufügen wollen,
so können Sie die Erstellung der Ziel-URL auf der Detailseite durch entsprechende TypoScript Konfiguration
vornehmen.

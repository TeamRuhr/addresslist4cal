Bekannte Probleme
=================

addresslist4cal sollte mit allen freigegebenen TYPO3 Versionen funktionieren.

Im `Bugtracker`_ befinden sich alle bekannten Probleme.
Wenn Sie denken das Sie einen Fehler gefunden haben, melden Sie ihn bitte auf Forge (falls es ihn nicht schon gibt)
und fügen Sie mindestens folgende Informationen hinzu um die Lösung so einfach wie möglich zu machen:

- Verständlicher Titel und Beschreibung: Bitte schreiben Sie in Englisch und beschreiben Sie wie das Problem
  reproduziert werden kann.

- TYPO3, cal, tt_address & addresslist4cal Versionen.

- Natürlich wäre ein Patch großartig, aber das ist kein Zwang.

.. _Bugtracker: http://forge.typo3.org/projects/extension-addresslist4cal/issues
